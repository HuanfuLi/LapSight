# Phase 3 Plan 01: Mobile GPS Probe / Demo Feed Slice Summary

**Provider-layer deterministic simulated GPS feed (LocationSampleProvider + SimulatedGpsProvider) with fixed-seed GpsFixtureLibrary covering all six D-04 scenarios, a GpsQualitySummary rollup, committed JSON fixtures, and a Drive Start/Stop Demo Feed control with an amber DEMO badge.**

## Performance

- **Duration:** ~28 min
- **Completed:** 2026-06-25
- **Tasks:** 3
- **Files modified:** 13 (12 created, 1 modified)

## Accomplishments
- `LocationSampleProvider` boundary that simulated and future real GPS both implement, so marking/timing flows consume one seam (D-03).
- `SimulatedGpsProvider`: deterministic, continuous feed with independent `start`/`stop`/`reset`/`nextSample`/`isRunning` lifecycle and an exposed active scenario id.
- `GpsFixtureLibrary`: fixed-seed builders for `clean-10-loop`, `minimum-5-loop`, `one-outlier-loop`, `noise-drift`, `dropped-low-frequency`, and `multi-session-best-candidate`, each fully `LocationSource.Simulated` (D-01/D-02/D-04/D-42).
- `GpsQualitySummary.from(samples)`: sample count, duration, average update rate, best/worst accuracy, degraded-sample count, and source set — empty-safe.
- Committed JSON payloads for all six scenarios under `composeResources/files/fixtures/`.
- Drive surface now starts/stops the provider feed, shows the amber `DEMO — simulated GPS` badge, live speed/accuracy/samples/rate, and blocks formal timing with "Mark a track first. Timing needs a saved start/finish line." (D-05/D-19/D-44).

## Decisions Made
- Provider boundary replaces only the GPS layer; no separate demo-only session flow (D-03).
- Continuous feed wraps with forward-advancing timestamps (D-05) rather than stopping after one pass.
- Fixtures use a self-contained LCG + fixed seeds (no kotlinx-serialization or platform RNG), keeping shared tests free of Compose/Android/iOS/Okio/serialization. JSON payloads are committed copies for external/debug tooling.
- Formal timing is a visibly disabled control with explanatory copy, not hidden, to satisfy D-19 while keeping the Drive surface honest.
