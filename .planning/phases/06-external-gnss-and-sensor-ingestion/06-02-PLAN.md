<task type="auto">
  <name>Task 1: RaceBox frame parser and synthetic builder</name>
  <action>
    Implement a small RaceBox frame parser and a test-only synthetic frame
    builder. The parser must handle complete frames, fragmented frames, multiple
    frames per buffer, checksum failure, unsupported message ids, no-fix frames,
    and valid fix frames. Map units explicitly into LapSight units: degrees,
    meters, meters/second, degrees heading, and accuracy meters. If the protocol
    documentation or clean-room synthetic corpus exposes accelerometer, gyro, or
    simple vehicle channels, parse them into optional telemetry records with the
    same source/protocol provenance; otherwise emit an explicit unsupported
    telemetry result instead of inventing fields.
  </action>
  <verify>
    <automated>./gradlew :shared:testAndroidHostTest --tests "*RaceBoxFrameParserTest*"</automated>
  </verify>
  <acceptance_criteria>
    - Valid synthetic RaceBox frames decode to expected external fix data.
    - Basic telemetry frames, when represented in the synthetic corpus, decode to
      optional telemetry metadata and never affect lap timing.
    - Fragmentation and burst tests are deterministic.
    - Malformed/checksum-failing frames are rejected without throwing.
    - Parser code is original and does not copy GPL/unlicensed implementations.
  </acceptance_criteria>
</task>
