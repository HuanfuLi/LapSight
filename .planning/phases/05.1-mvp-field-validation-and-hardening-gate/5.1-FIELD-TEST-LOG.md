## Per-session evidence template (D-48 minimal field notes)

Each valid session records:

| Field | Meaning |
|-------|---------|
| **Device** | Device name/model + label (Primary / Secondary) + OS version (D-07). |
| **Session ID / export file** | Saved session identifier and the export file path (JSON; GPX where available) (SESS-04/SESS-05). |
| **Lap count pass/fail** | Engine lap count vs manual/video tally — must be 100% correct, the HARD blocker (D-19/D-20). |
| **Replay result** | `SessionReplayDecoder.decodeForReplay` → `Decoded`, deterministic across runs, lap count matches (D-25/D-28). |
| **Video reference / result** | Start/finish video file; lap-time vs readable crossing ≤ 0.5 s target; unreadable crossings excluded (D-21/D-23). |

**Optional extras (D-49)** — fill ONLY on failure/anomaly: weather, mount position, lighting, temperature, battery, GPS Ready wait time, abnormal events.
