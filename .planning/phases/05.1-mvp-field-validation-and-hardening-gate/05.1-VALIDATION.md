- **After every task commit:** Run `./gradlew :shared:testDebugUnitTest`
- **After every plan wave:** Run full shared suite + `./gradlew :androidApp:assembleDebug`
- **Before `/gsd:verify-work`:** Full suite green AND all five evidence artifacts complete
  (`5.1-CODE-REVIEW.md`, `5.1-UAT.md`, `5.1-UI-REVIEW.md`, `5.1-FIELD-TEST-LOG.md`, `5.1-GO-NOGO.md`)
