**Core pattern to build (composes the three seams; `ReplayRunner.run` is the consume side, ReplayRunner.kt:54-60):**
```kotlin
// NEW — pure, commonMain, no platform deps. Bad bytes are DATA not exceptions
// (mirror JsonExportService's Corrupt handling and WrongCoursePreflight.Unavailable).
fun decodeForReplay(exportedBytes: ByteArray, config: LapEngineConfig = LapEngineConfig()): ReplayResult {
    val payload = FileSessionStore.canonicalJson
        .decodeFromString(TimingSessionPayloadV1.serializer(), exportedBytes.decodeToString())
    val samples = payload.samples.map { it.toModel() }
    val course = courseFromTrack(
        payload.session.startFinish, payload.session.sectors, payload.session.direction,
    ) ?: error("payload has no confirmed start/finish")
    return ReplayRunner(course, config).run(samples)   // ReplayRunner.kt:54
}
```

## Shared Patterns

### Clean-room engine boundary (AGENTS.md non-negotiable)
**Source:** AGENTS.md ("Keep lap engine logic independent from UI and platform APIs"); enforced throughout `shared/src/commonMain/.../shared/lap`, `ghost`, `session`.
**Apply to:** every NEW algorithmic file (SessionReplayDecoder, ReadyGate, raw-recording seam, fixture). They must import zero Compose/Android/iOS/Okio symbols and be exercised from `commonTest`. The decoder must reuse `LapEngine`/`ReplayRunner`, never a second engine.

### Pure-engine + injected-clock determinism
**Source:** `session/SessionController.kt:51` `private val now: () -> Long`; `ui/DriveMarkingController.kt:131`; `lap/ReplayRunner.kt` (fresh engine per run).
**Apply to:** the determinism suite and any test driving the controllers — construct with a fixed `now` + `InMemorySessionStore`, feed fixture samples, assert structural equality. Reserve tolerances only for derived doubles; use exact `Long`-millis equality for times (RESEARCH Pitfall 6).

### Sealed-result + "bad input is data" error handling
**Source:** `session/WrongCoursePreflight.kt:51-67` (`CoursePreflightResult` + `Unavailable(reason)`); `storage/LoadResult.Corrupt`; `export/JsonExportService.kt:37-38` (`ExportFailedException`).
**Apply to:** ReadyGate (`NotReady(reasons)`), SessionReplayDecoder (typed corrupt path), raw-recording seam. Never throw on malformed GPS/JSON in core paths (V5 input-validation, RESEARCH Security Domain).

### Serializable DTO ↔ domain mappers (never re-parse)
**Source:** `session/SessionModels.kt` `*.toDto()`/`*.toModel()` (lines 142-163, 254-274, 541-643); `session/TimingSessionRecorder.kt` `*.toDomain()` (lines 30-50).
**Apply to:** the decoder (reuse `LocationSampleDto.toModel()`), the raw-recording export (reuse `LocationSample.toDto()`). Reuse `FileSessionStore.canonicalJson` so encode/decode stay byte-symmetric.

### Deterministic seeded fixtures
**Source:** `fixtures/GpsFixtureLibrary.kt` `Lcg` + `ovalSession` + `scenario(id)` registry.
**Apply to:** the new degradation fixture and any recorded field-export fixture promoted into the suite (RESEARCH: the five field JSON exports become permanent replay fixtures, D-12/D-28).
